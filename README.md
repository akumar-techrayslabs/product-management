# Product Management
This project is a simple Product Management System with basic CRUD Operation 
in which you can add, list, delete or edit a product, for storing data it uses browser's local storage.


## Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/akumar-techrayslabs/product-management.git
   ```

2. **Install dependencies**:
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Compile TypeScript files**:
   ```bash
   tsc .\src\scripts\src\*.ts
   ```

4. **Open the application**:
   Open `index.html` in your web browser to view the application.
